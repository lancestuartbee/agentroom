# 翻译句示例

Phase 2 详细示例表。内部术语首次出现时，紧跟一个"翻译句"。

| 内部黑话 | 翻译句示例 |
|---------|-----------|
| 蒸汽马车 | 用旧世界的形态想象新物种——给 agent 分"产品经理/开发/测试"岗位 |
| 能力画像 | 不问 agent 是什么角色，问它擅长什么、什么条件下容易出错 |
| 方差吸收 | 单个模型犯了蠢，系统里有另一个模型接住——用户感觉不到波动 |
| harness | 把多个 AI 组织在一起的运行框架——相当于操作系统，模型是应用 |

**规则：一个词在内部讨论出现 >3 次但外部文档出现 0 次 → 黑话，必须翻译。**
